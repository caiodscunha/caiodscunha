# Caio Cunha
Desenvolvedor Backend e Desenvolvedor de IA com experiência em Java, Spring Boot, Python, TypeScript e
FastAPI, atuando na construção de APIs REST escaláveis, integrações e autenticação de usuários. Realizei
fine-tuning de modelos de IA com PyTorch e implementei soluções com MongoDB, PostgreSQL, Firebase (Cloud
Functions, Firestore) e Oracle SQL. Aplico testes automatizados (JUnit), versionamento com Git e
metodologias ágeis (Scrum) em projetos colaborativos. Complemento meu perfil técnico com experiência prática
em análise de dados com Pandas e criação de dashboards com Power BI, DAX e Power Query. Cursando Engenharia
de Software na PUC-Campinas (2024–2027) no período matutino, com atuação em projetos reais em parceria com
a Venturus.
